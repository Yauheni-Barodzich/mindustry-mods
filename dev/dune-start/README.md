# Дюна: Старт (`dune-start`)

Контент-мод для **Mindustry v159+**: ранняя наземная добыча и лёгкая оборона без энергии в духе Дюны.

Сборка zip и общая установка пакета — в [корневом README](../../README.md).

| | |
|--|--|
| ID мода | `dune-start` |
| Версия | см. `mod.hjson` |
| Исходники | `dev/dune-start/` |
| Артефакт | `dist/dune-start.zip` → `mods/` и `server-mods/` |

```powershell
.\dev\dune-start\build.ps1
```

---

## Простая мастерская

| Параметр | Значение |
|----------|----------|
| ID блока | `dune-start-rover-factory` |
| Тип | `UnitFactory` |
| Размер | 3×3 |
| HP / броня | 550 / 1 |
| Энергия | не нужна |
| Постройка | **70 меди** |
| Исследование | `core-shard` (×0.2) |
| Выпуск | спереди (`out`-спрайт) |

### Планы производства

| Юнит | Время | Стоимость | Лимит на команду |
|------|-------|-----------|------------------|
| Вездеход-сборщик | **4 с** (240 тиков) | **10 меди** | **3** |
| Дрон-камикадзе | **3 с** (180 тиков) | **15 свинца** | **2** |

Лишний выпуск отменяется, стоимость плана возвращается в ядро, прогресс завода сбрасывается.

## Вездеход-сборщик

| Параметр | Значение |
|----------|----------|
| ID юнита | `dune-start-sand-rover` |
| Тип | `tank` (наземный, не ходит сквозь стены) |
| HP / броня | 200 / 2 (личный maxHP от сида, см. ниже) |
| Скорость базы | 0.34 |
| Трюм | **200** (общий `itemCapacity`, без разброса) |
| Добыча | `mineTier` 2, `mineSpeed` 6, `mineRange` 55 |
| Команды | `move`, `mine` (по умолчанию `mine`) |
| Разгрузка | штатный **MinerAI** → ближайшее **ядро** (`range` 50) |

### Поведение

- Добыча и возврат на ядро — стандартные **CommandAI / MinerAI**.
- Скрипт: лимит, трейты, HUD, пыль, скорость от загрузки трюма, износ.
- Список руд (`mineItems`) — по `mineTier` и наличию на карте.

### Уникальность (сид `unit.flag`)

| Трейт | Разброс |
|-------|---------|
| Окрас | случайный HSV |
| Скорость хода | ±10% |
| Скорость добычи | ×0.8 … ×2.0 |
| Прочность (maxHP) | ×0.65 … ×1.2 |
| Стартовая цель руды | из доступных на карте |

Трюм у всех одинаковый (**200**). Износ ≈ **14% maxHP** за полный трюм при выгрузке.

## Дрон-камикадзе

| Параметр | Значение |
|----------|----------|
| ID юнита | `dune-start-kamikaze-drone` |
| Тип | `flying` |
| AI | `SuicideAI` + скрипт обороны базы |
| HP | 90 |
| Скорость | 2.4 |
| Взрыв | splash **110** / радиус **48** (`killShooter` + `shootOnDeath`) |
| Команды | `move` (цель атаки — через скрипт/`commandTarget`) |
| Лимит | **2** на команду |
| Стоимость | **15 свинца** |

### Поведение

- Скрипт ищет врагов в радиусе **48 клеток** от **своего** ядра и отдаёт `attack`.
- Нет угрозы у базы — `move` обратно к ядру (не летит на вражеское ядро сам).
- При сближении штатный suicide-weapon взрывает дрона.
