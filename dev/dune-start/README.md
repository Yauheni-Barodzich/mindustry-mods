# Дюна: Старт

Каркас контент-мода Mindustry для ранней техники и блоков в стиле Дюны.

## Структура

```
dune-start/
├── mod.hjson          # метаданные мода
├── content/
│   ├── items/         # ресурсы (hjson)
│   ├── blocks/        # постройки, турели, дробилки…
│   ├── units/         # техника / юниты
│   └── liquids/       # жидкости (по необходимости)
├── sprites/
│   ├── items/
│   ├── blocks/
│   └── units/
├── bundles/           # локализация EN / RU
└── sounds/
```

Имя файла без расширения = id контента. В игре id будет `dune-start-<имя-файла>`.

## Установка (пока без сборки)

Скопировать папку `dune-start` в `mods/` (или zipнуть содержимое и положить zip туда).

## Сейчас в моде

| Контент | Стоимость | Примечание |
|---|---|---|
| `rover-factory` | 50 меди | **Гараж вездеходов**, 3×3, без энергии |

| `sand-rover` | 5 меди, ~5 с | вездеход-сборщик, лимит **3** на команду |

Лимит юнитов — `scripts/main.js` (лишний юнит снимается, медь возвращается на ядро).

## Дальше

Добавлять hjson в `content/*` и спрайты с тем же именем в `sprites/*`.
